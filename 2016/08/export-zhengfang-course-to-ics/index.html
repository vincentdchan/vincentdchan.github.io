把正方系统的课表导出成 ics 文件
Posted at 2016-08-14

文章的开始，先上一张图，打开 Siri，问 Siri：明天有什么安排？

为什么 siri 知道我的课表？因为我把课表导入了 iPhone 的日历中去了。

因为不想再使用超级课程表等软件，加上华农宝不怎么好看，而且很麻烦的缘故，自己写了一个工具，可以把课程表导出到ics 格式，可以把这个文件导入到系统的软件当中，比如iPhone自带的日历, Outlook, Google Calendar 等等你常用的日历工具……导入到 iPhone 或者 Android 手机自带的日历之后，无论是查看，通知甚至类似 Siri，小娜的各种语音助手进行查询，都会华农宝方便很多。

另外，这个插件是安装在你的电脑上进行导出，它的代码完全开放，托管在 Github，你无需担心密码泄漏等安全问题，相反华农宝使用你的密码登录教务系统，会使人多了一层疑虑。

目前来说，你可以通过使用 Chrome 或者 Tampermonkey 插件来使用我做的插件进行导出。

方法一：Tampermonkey（油猴）插件

Tempermonkey 插件支持的浏览器有:

  • Google Chrome
  • Firefox
  • Edge

以上浏览器安装 Tampermonky 插件，安装方法自行搜索。

安装完成之后，去到这个链接点击安装，安装成功之后登入你的正方系统，去个人课表导出页面设置好日期之后导出就好了，然后大功告成。

方法二：Chrome插件

下载下面的 Chrome 插件（仅华农的正方系统适用），安装方法请自行搜索

Github 地址: https://github.com/vincentdchan/ZhengFangToolkit

下载 Chrome 插件，并安装，安装方法请自行搜索，安装完成后，登入正方系统，进入个人课表，选择好正确的学年和正确的学期，检查一下开学的第一个星期一是否正确，不然的的话生成的日历全部都会错误的。

检查完之后点击导出课程，最后会出现一条下载链接，下载那个 ics 文件就行了。 然后把 ics 文件导入到各种软件中去，可以看看效果图
