转换网易 ncm 格式到 mp3
Posted at 2019-05-08

不知道什么时候开始，网易下载的歌曲都不用以前的 mp3 格式，而采用了自家的 ncm，说真的，比较恶心，平时自己想要剪个视频用个配乐都很麻烦，搜了很久很多工具，最后搜到了这个：ncmdump。

首页提供了 Windows 和 Linux 的使用方法，mac 下面使用也很简单，先用 brew 安装 taglib，然后 make 即可：

$ git clone https://github.com/anonymous5l/ncmdump
$ brew install taglib
$ make
